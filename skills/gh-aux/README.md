# gh-aux Skills

Skill for [gh-aux](https://github.com/canoypa/gh-aux) — a GitHub CLI extension.

This skill covers GitHub operations not available in the standard `gh` CLI.

## Installation

```sh
gh skill install canoypa/gh-aux gh-aux
```

For the full list of available commands, see the [gh-aux README](https://github.com/canoypa/gh-aux#commands).
